 <h2>Description of Dataset</h2>
    <h3>The pickle data contains a dictionary with 4 key-value pairs: </h3>
    <ul>
      <li><h4><h3>'features'</h3> is a 4D array containing pixel data of the traffic sign images, (number of  examples, width, height, channels).</h4></li>
      <li><h4><h3>'labels'</h3> is a 1-Dimensional array containing the class id of the traffic sign. The file signnames.csv contains id -> name mappings for each id.</li></h4>
       <li><h4><h3>'sizes'</h3> is a list containing tuples-(width, height) representing the original width and height.</li></h4> 
       <li><h4><h3>'coords'</h3> is a list containing tuples, (x1, y1, x2, y2) representing coordinates of a bounding box around the sign.</li></h4> 
    </ul>
